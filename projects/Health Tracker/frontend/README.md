# Health Tracker - React Frontend

A modern React.js frontend for the Health Tracker application with improved dropdown functionality and component-based architecture.

## Features

- **React Components**: Modular, reusable components for better maintainability
- **Custom Dropdowns**: Fully functional dropdowns with proper state management
- **Calendar Interface**: Interactive calendar with date selection and visual indicators
- **Form Management**: Comprehensive health tracking form with validation
- **API Integration**: Seamless communication with the Flask backend
- **Responsive Design**: Mobile-friendly interface

## Components

- `App.js` - Main application component with state management
- `Calendar.js` - Calendar rendering and date selection
- `HealthForm.js` - Health tracking form with all input fields
- `CustomDropdown.js` - Reusable dropdown component with proper click handling

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm start
   ```

3. The app will open in your browser at `http://localhost:3000`

## Backend Requirements

Make sure the Flask backend is running on `http://localhost:5002` before using the application.
